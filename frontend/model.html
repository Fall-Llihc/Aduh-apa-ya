Pipeline

  1. Preprocessing — lowercase, hapus URL/angka/non-alfabet, tokenisasi NLTK, stopword removal (NLTK + 29 custom BBC stopwords), lemmatization WordNet.
  2. TF-IDF vectorization — ngram_range=(1,2), max_features=20,000, sublinear_tf=True, min_df=2. Konfigurasi ini hasil eksperimen sistematis pada 4 parameter kunci.
  3. Stacking Classifier — 3 base estimators (LinearSVC, ComplementNB, Logistic Regression) digabung dengan meta-learner Logistic Regression. Setiap base model di-tune via GridSearchCV terlebih dahulu.
  4. Cross-validation — semua evaluasi menggunakan 5-fold Stratified CV dengan Pipeline yang benar untuk mencegah data leakage.

Why this approach

Model linear (LinearSVC, Logistic Regression) konsisten mengungguli model tree-based (Random Forest, Extra Trees) untuk data TF-IDF yang sparse dan berdimensi tinggi. Stacking ensemble menggabungkan kekuatan komplementer dari tiga model dengan pola error berbeda, sehingga prediksi akhir lebih robust.

Limitations

  • Dataset hanya BBC News (gaya penulisan Inggris British) — generalisasi ke media lain perlu diuji.
  • Hanya 5 kategori; topik di luar lima ini akan dipaksa masuk salah satunya.
  • Tech ↔ business overlap secara tematik — artikel teknologi korporat sering ambigu.
  • Input non-English diterjemahkan dulu ke English sebelum diklasifikasi.
